Projects

A selection of research projects and software tools I've built during my graduate and undergraduate work.

Neuromorphic Camera Modeling

Developed an analytical statistical model for event-based cameras that jointly describes signal-driven and noise-induced events. Designed experimental methods to infer unknown physical sensor parameters from recorded event streams.

Python Statistical Modeling Sensor Physics

Deep-Learning Image Reconstruction

Trained U-Net architectures with self-attention to reconstruct still images from event-camera noise, using reconstructions as a validation signal for the underlying physical and statistical model.

Deep Learning U-Net Self-Attention Python

Schrödinger Dynamics Simulator

Built and maintained an open-source numerical simulation suite for time-dependent Schrödinger dynamics with non-trivial, time-dependent boundary conditions.

Python Numerical Methods Quantum Mechanics

Quantum Randomness Testing

Investigated the statistical randomness of bit strings generated by IBM quantum computers. Applied probability-theoretic tests to compare quantum outputs with classical pseudo-random generators, with implications for cryptography.

Phase-Modified Oracle Discrimination

Developed a quantum algorithm that distinguishes phase-modified unitary oracles using a single-qubit measurement, contributing to quantum information theory foundations.

Quantum Algorithms Quantum Information

Spinning Optomechanical Resonator
